# CLAUDE.md - プロジェクト設定

## 基本方針
このプロジェクトはClaudeCodeを使用した日本語環境での開発です。

## 言語設定
- **回答言語**: 日本語のみ
- **コメント**: 日本語で詳細に記述
- **変数名**: 英語（業界標準に準拠）
- **関数名**: 英語（camelCase形式）

## コーディング規約
- インデント: スペース2文字
- コメント: 処理の意図を日本語で詳細に説明

## 禁止事項
- 機密情報の外部送信
- 本番環境への直接操作
- 未テストコードの本番適用

## 開発フロー
1. 要件定義（日本語）
2. 設計書作成（日本語コメント付き）
3. コード実装
4. テスト実施
5. ドキュメント更新
